
-타입이란?
값을 저장하기 위한 여러가지 자료형
- 타입을 언제 사용해야 되는지?
내가 원하는 데이터를 저장할 때 사용

-클래스란?
객체의 틀- 묶음 : 서로 다른 타입(기본타입[8], 참조타입 )틀
객체에 대한 세부 정보


객체 설계도 : 서로 다른 타입들의 집합

-클래스를  언제 사용해야 되는지? 
객체를 만들기 위해 , 반복되는 데이터들을 보다쉽게 정리하기 위해

-객체란?
속성과 메소드를 실체화된 것으로 표현한 것

- 객체는 언제 사용해야 되는지?
속성과 메소드를 하나의 실체로 만들어 디지털화 하기 위해


실체를 만들어 현실에 있는 모든것을 디지털화 하기위해

-멤버변수 란?
클래스 영역에서 선언된 변수

-멤버변수는 언제 사용해야 되는지??
클래스안에서 변수를 선언할 때
    -왜? 실행함수 안에 값을 넣어서 기능 활성화 할 변수를 선언할 때



-생성자란?
멤버변수나 상수를 초기화
-생성자는 언제 사용해야 되는지?
멤버변수값을 지정하고 인스턴스 값을 초기화


-메소드/함수란? 객체들의 상호작용
클래스의 기능을 하는 코드를 묶어서 나타내는 것(상호작용)
-메소드/함수란 언제사용해야되는지?
필요한 기능을 실현시킬 때

-매개변수란?
함수내에서 사용할 변수를 호출시 전달받을 값들
-매개변수는 언제사용해야되는지?
함수안에서 변수를 호출하여 변수 안의 값을 전달할 때

-반환타입이란?
메소드 실행 후 반환/리턴할 값의 타입.
-반환/리턴이란?
 함수 내에서 결과를 호출했던곳으로 전달할 값
-반환/리턴을 언제 사용해야되는지?

메소드가 실행 후 결과를 호출한 곳에 돌려받고자 할 때

반환값이 없는 경우에는 void 씀
