



    주제 : 용어 정의하기
       1. 객체 란 ?
       2. 클래스 란 ?
       3. 멤버변수/필드/속성/특성 이란?
       4. 지역변수 와 멤버변수의 차이점을 서술하시오.

    내 답:
           1. 객체란?
           : 객체는 일상생활에서 쓸 수 있는 실체화된 모든 것을 디지털화 한 것이다 .
               - 자바에서 객체 표현은  클래스명 변수명 = new 클래스명();으로 표현한다.
            객체의 멤버(필드/메소드) 접근/참조하는 방법
            : 객체.멤버변수 (객체내 멤버변수/필드에 값 대입)

            2. 클래스란?
            : 객체의 틀/설계도 : 서로 다른 타입들의 집합이다.
                -클래스는 객체의 상태를 나타내는 필드와 객체의 행동을 나타내는 메소드로 구성된다.
                    -메소드: 어떠한 특정 작업을 수행하기 위한 명령문의 집합.(객체와 상호작용)

             3. 멤버변수/핃드/속성/특성 이란?
                1. 멤버변수
                    -클래스 영역에서 선언된 변수 (static)
                   - 지역변수: 메소드/생성자  안에서 선언된 변수 (인스턴스 변수, 값이 고정되지 않아서 각 객체마다 다른값)
                2. 필드
                    -클래스를 구성하는 요소 중 하나로 클래스 내부 멤버. (어떠한 데이터를 저장하기 위한 역할을 담당
                    -클래스 내부의 생성자와 메소드 밖에 정의가 됨.)
                3. 속성
                    - 클래스내에서 있는 변수, 객체의 상태.
                        -변수와 속성의 차이점
                        : 변수는 해당 메소드와 생성자에서만 사용되고, 메소드와 생성자가 종료됨과 동시에 메모리가 소멸됨.
                          (필드는 클래스 안의 전체 메소드와 생성자에서 사용되고 객체가 소멸될때까지 존재.)
                4. 특성
                    - 상속 : 속성과 메소드가 약간 다른 객체를 생성하기 위한 방법 (기존에 클래스로부터 모든 속성과 메소드를 상속받고
                                                                        새로운 클래스 생성가능)
                    -캡슐화 : 객체를 숨겨야 하는 정보와 공개해야하는 정보를 구분하여 객체로부터 정보를 은폐하는 것.

                    -다형성 : 하나의 인터페이스를 이용해 서로 다른 구현 방법을 제공하는 것 (하나의 이름으로 사용되지만 여러 개의 구현을
                                                                                   가지는 개념)-객체들의 호환

                -즉 객체를 생성하고 값을 대입하는 것을  멤버변수/핃드/속성/특성라고 한다.
                5. 지역변수와 멤버변수의 차이점?
                    - 지역변수는 메소드 내부에서 선언하여 메소드 안에서만 사용이 가능하다.
                      반면에 멤버변수는 클래스에서 선언하여 아무도 해당 인스턴스를 참조하지 않으면 가비지 컬렉터(GC)가 수거해감.
